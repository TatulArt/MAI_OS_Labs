# LR 1 - Управление процессами и каналы

## Вариант 21

Родительский процесс создает два дочерних процесса. Пользователь вводит строки произвольной длины в родительский процесс, которые распределяются по правилу:
- Нечетные строки отправляются в pipe1
- Четные строки отправляются в pipe2.

Дочерние процессы инвертируют строки.

---

## Компиляция и запуск

```bash
gcc -o client client.c
gcc -o server server.c
./client
```
